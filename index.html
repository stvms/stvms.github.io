Mixture of Experts - AI driven MultiView Node infinite Canvas

Infinite Canvas with MultiView Nodes and the Custom Context Menu

A node-based workspace designed to use various Mixture-of-Experts models dynamically in an interconnected way. Each model has its own specialization and allows quick generation. Additional focus in this project lies on local and offline generation. Supports Text-to-Image, Image-to-Image, Image-to-Text and Text-to-Text workflows, building a closed loop of chainable operations. Inspired by Flora and Fauna AI.

The results are endless.

Technologies

Frontend: React, Reactflow, Backend: Python
For the API calls ComfyUI is used as the Engine. With a custom 4-in-1 workflow for the different models that are switched via the backend script.

Models: SD 1.5, SDXL (text-to-image generation), DeepSeek - Janus Pro (image understanding), Mistral-7B (unimodal language model). All models are around 4GB, tested on a GTX 1060 6 VRAM, focusing on low hardware requirements and constrained resources. Easily scalable on more powerful hardware.

Interactive Scribbling

Using a sketch as the starting point for controlled image generation with SD 1.5

A project designed to iterate your own sketch with the help of machine learning. Rather than using only prompts to create an image, the user is encouraged to sketch his vision and apply different stylings to it. This allows the generated image to better align to the vision of the creator.
Furthermore the goal is to incorporate AI generated workflows in iterative steps, instead of common 'prompt and go' generations. By allowing the user to draw, the image generation is guided by the actual drawing and not only relying on prompts. Bringing AI workflows more in line with the vision of the artist.

Technologies

Python and Gradio for the frontend. ComfyUI as the backend image generation.
A custom workflow is specialized for drawings (Control Net).

Real-Time Capture Anywhere & Generate

Real time capturing while sketching, prompts the generation engine in parallel after every stroke

Extending the project Interactive Scribbling by focusing on quicker generation and a more flexible usage across the board, comes Real-Time Capture Anywhere & Generate. Inspired by the concept of LCM loras and flowtyone's realtime lcm canvas, which allows very fast generation of drawings even on lower tier hardware.
Capture Anywhere & Generate has the added benefit, to not rely on the gradio built-in Image Editor for the drawing canvas in the original prototype.

Instead with Capture Anywhere the user can select his own preferred application of choice. Photoshop, Blender etc. Monitoring and instantly generating an image, without leaving his workplace.

Similar to Interactive Scribbling, this approach brings user creativity even closer to an iterative process with AI involved. Giving more control to the individual and placing AI as a 'Tool' and not a 'Creative Director'.

Technologies

Python and Gradio for rapid prototyping a User Interface. The desktop capturing can be configured via the 'Set Region & Capture' button located at the left toolbar.
Models: LCM Lora SD 1.5 with Lykon Dreamshaper-7

Showcase

Note: LCM Lora and real-time monitoring allows for very fast results, around 2 seconds on a GTX 1060 6GB. Especially when capturing after every stroke making the AI workflow near instant or in other words running in parallel. Tip: keeping the step count at 1 or 2, renders the AI generated image as close as possible to the actual sketch, which again should better align with the vision of the artist.
